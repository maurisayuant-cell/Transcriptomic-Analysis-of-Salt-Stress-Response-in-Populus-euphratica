# Transcriptomic Analysis of Salt Stress Response in Populus euphratica

## Overview
This project investigates gene expression changes induced by severe salt stress (600 mM NaCl) in Populus euphratica using microarray data (GSE52305).

## Methods
- Differential expression analysis using limma
- Contrast: Salt600 vs Control
- Significance threshold: adjusted p-value < 0.05
- GO term frequency analysis

## Results
- 7218 significantly differentially expressed genes
- 3023 upregulated genes
- 4195 downregulated genes
- Enrichment in transcription regulation, carbohydrate metabolism, transport, and cell wall modification processes

## Conclusion
Severe salinity induces large-scale transcriptional and metabolic reprogramming in Populus euphratica, highlighting coordinated stress adaptation mechanisms.
